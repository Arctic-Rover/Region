Аннотация

Проект «АВТОНОМНЫЙ ИССЛЕДОВАТЕЛЬСКИЙ РОБОТ ДЛЯ АРКТИКИ — АРКТИКАХОД» разработан для комплексного и автономного проведения научных исследований в суровых климатических условиях Арктики. Робот предназначен для мониторинга льда, флоры и фауны, и передачи данных учёным без риска для жизни человека.
